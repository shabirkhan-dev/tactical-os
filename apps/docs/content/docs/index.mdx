Welcome to the official documentation for the Starter monorepo.

This project combines modern tooling and a production-minded structure so you can ship web,
mobile, and API apps from one workspace.

## Why this school-os

- Monorepo by default with Turborepo workspaces (`apps/*`, `packages/*`)
- Bun-first package and task management
- Shared TypeScript and UI packages across apps
- Built-in quality gates with Biome and Lefthook
- Multi-platform app setup (Web, Mobile, Nest API, Docs, Rust)
