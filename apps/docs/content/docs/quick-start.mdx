---
title: Quick Start
description: Get the Starter Kit running in minutes.
---

## Prerequisites

- Bun (latest stable)
- Git
- Node-compatible system dependencies for Next.js and Expo

## 1) Install dependencies

```bash
bun install
```

## 2) Install git hooks (recommended)

```bash
bun run prepare
```

This enables Lefthook checks for formatting, linting, type checks, and commit hygiene.

## 3) Run the workspace

```bash
bun run dev
```

Turbo will run relevant app/dev tasks across the monorepo.

## 4) Open apps

- Web app: `apps/web`
- Mobile app: `apps/mobile`
- API app: `apps/hono-api`
- Docs app: `apps/docs`

## 5) Validate quality before pushing

```bash
bun run lint
bun run format
bun run typecheck
```

If your change only touches one app, you can still run checks at the root to keep CI consistent.
